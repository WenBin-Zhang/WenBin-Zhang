JavaScript中i++和++i的区别

发表于 2020-05-11 分类于 JavaScript ， operator 阅读次数：

一、++ i 和 i ++简单介绍

++ 运算符又称自增运算符，功能为将操作数自加 1。

++ i 被称为前加加，++ 后面的变量执行自加操作，执行顺序为：先执行变量的自加操作，再引用变量的值。

i ++ 被称为后加加，++ 前面的变量执行自加操作，执行顺序为：先引用变量的值，在执行自加操作。
