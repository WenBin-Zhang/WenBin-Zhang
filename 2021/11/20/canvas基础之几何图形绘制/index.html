canvas基础之几何图形绘制

发表于 2021-11-20 更新于 2021-11-21 分类于 canvas 阅读次数：

Canvas 是 HTML5新增的，一个可以使用脚本（通常为 JavaScript）在其中绘制图像的 HTML5 元素。它可以用来制作照片集或者制作动画，甚至可以进行实时视频处理和渲染。
Canvas 是由 HTML 代码配合高度和宽度属性而定义出的可绘制区域。JavaScript 代码可以访问该区域，类似于其它通用的二维 API, 通过一套完整的绘图函数来动态生成图形。

一、Canvas基本使用

1.1、canvas 元素

canvas 只有两个可选的属性 width、height属性，默认width = 300、height = 150, 单位为px。也可以通过css属性设置画布的宽高，但是通过css设置的宽高必须和默认的画布宽高比例保持一致，否则画布会发生扭曲。所以最好不要使用css属性设置画布的宽高。

由于canvas是HTML5新增的标签，所以在一些低版本的浏览器会出现兼容性问题（IE9以前的IE浏览器）。所以需要准备一定的替换内容。

浏览器如果支持canvas标签，则不会执canvas标签内部的内容，如果不支持canvas标签，则会显示标签里面的内容。

2.2、渲染上下文(The Rending Context)

<canvas> 会创建一个固定大小的画布，会暴露一个或多个渲染上下文(画笔)，使用渲染上下文来绘制和处理要展示的内容。主要用到的是2D渲染上下文，除此之外还有3D渲染上下文。

同样的如果浏览器不支持canvas标签，那么在脚本中获取2d上下文对象会失败，甚至可能会报错影响正常的脚本执行，所以在js中获取上下文对象也需要做兼容性判断。
