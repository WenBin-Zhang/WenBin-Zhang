canvas基础之几何图形绘制

发表于 2021-11-20 更新于 2021-11-21 分类于 canvas 阅读次数：

Canvas 是 HTML5新增的，一个可以使用脚本（通常为 JavaScript）在其中绘制图像的 HTML5 元素。它可以用来制作照片集或者制作动画，甚至可以进行实时视频处理和渲染。
Canvas 是由 HTML 代码配合高度和宽度属性而定义出的可绘制区域。JavaScript 代码可以访问该区域，类似于其它通用的二维 API, 通过一套完整的绘图函数来动态生成图形。
本文内容借鉴于https://www.runoob.com/w3cnote/html5-canvas-intro.html

一、Canvas基本使用

1.1、canvas 元素

canvas 只有两个可选的属性 width、height属性，默认width = 300、height = 150, 单位为px。也可以通过css属性设置画布的宽高，但是通过css设置的宽高必须和默认的画布宽高比例保持一致，否则画布会发生扭曲。所以最好不要使用css属性设置画布的宽高。

由于canvas是HTML5新增的标签，所以在一些低版本的浏览器会出现兼容性问题（IE9以前的IE浏览器）。所以需要准备一定的替换内容。

浏览器如果支持canvas标签，则不会执canvas标签内部的内容，如果不支持canvas标签，则会显示标签里面的内容。

2.2、渲染上下文(The Rending Context)

<canvas> 会创建一个固定大小的画布，会暴露一个或多个渲染上下文(画笔)，使用渲染上下文来绘制和处理要展示的内容。主要用到的是2D渲染上下文，除此之外还有3D渲染上下文。

同样的如果浏览器不支持canvas标签，那么在脚本中获取2d上下文对象会失败，甚至可能会报错影响正常的脚本执行，所以在js中获取上下文对象也需要做兼容性判断。

二、绘制形状

2.1 栅格（grid）坐标空间

如下图所示、canvas元素默认被网格覆盖。通常来说网格中的一个单元相当于canvas元素中的一个像素。栅格的起点为canvas元素的左上角，坐标为（0，0）。canvas里所有的元素（图形）的位置都相对于原点进行定位。所以图中蓝色正方形左上角的坐标为距离左边（x轴）x像素，距离上边（y轴）y像素，坐标为（x,y）。

2.2 绘制矩形

canvas 只支持一种原生的图形绘制：矩形。绘制其它图形的需要通过路径(path)来实现。canvas 提供了很多路径生成方法用来实现复杂的图形绘制。

canvas 提供了三种方法绘制矩形：

  • 1、fillRect(x, y, width, height): 绘制一个填充的矩形。
  • 2、storkeRect(x, y, width, height): 绘制一个矩形的边框。
  • 3、clearRect(x, y, width, height): 清除指定区域的矩形区域，然后这块区域会变得完全透明。

说明：这三个方法的参数完全相同。

  • x, y: 指的是矩形的左上角的坐标。（相对于canvas的坐标原点-canvas元素的左上角）
  • width, height: 指的是绘制的矩形的宽和高。

三、绘制路径（path）

图形的基本元素是路径，除了矩形其它的图形需要通过路径来完成绘制。路径是通过不同颜色和宽度的线段或曲线相连形成的不同形状的点的集合。 路径都是闭合的，使用路径绘制图形需要一些额外的步骤：

  1. 创建路径起始点
  2. 调用绘制方法去绘制出路径
  3. 把路径封闭
  4. 路径确定以后，通过描边或填充路径区域来渲染图形

使用到的方法：

  1. beginPath() 表示开始绘制路径，图形绘制命令会被指向到路径上生成路径
  2. moveTo(x, y) 把画笔移动到指定的坐标（x，y），相当于设置路径的起始点坐标
  3. lineTo(x, y) 绘制一条从当前位置到指定位置（x, y）的直线，这里并不是直接绘制，真正的绘制要通过 stroke()、fill()等方法来实现。
  4. closePath() 闭合路径，闭合之后图形绘制命令又重新指向到canvas上下文中
  5. stroke() 通过线条来绘制图形轮廓，此方法不会自动闭合
  6. fill() 填充路径的内容区域生成实心图形，此方法会自动闭合

3.3 绘制圆弧

共有两个方法可以绘制圆弧：

  1. arc(x, y, r, startAngle, endAngle, anticlockwise):
    以（x, y）为圆心，以 r 为半径，从startAngle弧度开始到endAngle弧度结束。anticlockwise 是布尔值，true表示逆时针，false表示顺时针（默认是顺时针）。

    注意：

    1. 这里的度数都是弧度
    2. 0弧度是指的 x 轴的正方向

  2. arcTo(x1, y1, x2, y2, radius): 根据给定的控制点和半径画一段圆弧，最后再以直线连接两个控制点。
